Setup Database

Because the site is securely hosted on GitHub Pages, we use a free JSON bin to power the tracker.

  1. Go to JSONBin.io and create a free account.
  2. Click + Create New to make a new empty Bin. Paste `{}` inside it and hit Save.
  3. Click the Settings (gear) icon on that bin and switch it to Public.
  4. Copy the Bin ID (from the URL or settings) and your Master API Key (from the API Keys section).
